# 🐍 Python Notes — IITM BS

Week-by-week Python code notes from the **Programming in Python (BSCS1002)** course of the IIT Madras BS in Data Science & Applications program, taught by Prof. Sudarshan Iyengar.

Every concept from the lectures is written up as runnable Python code — no PDFs, no screenshots. Just `.py` files you can open, read, and actually run.

---

## 📚 About the Course

**Programming in Python** is the first formal programming course in the IITM BS program. It covers Python from scratch with a problem-solving focus — from basic syntax all the way through functions, recursion, collections, files, and modules.

- **Instructor:** Prof. Sudarshan Iyengar (Head of CSE, IIT Ropar)
- **Course ID:** BSCS1002
- **Level:** Foundational

## 🗓️ Week-by-Week Topics

| Week | Main Topic | Sub-topics Covered |
|------|---|---|
| **1** | Introduction to Python | Variables, literals, data types, expressions, operators, strings (indexing, slicing, methods), basic I/O, debugging |
| **2** | Conditionals | `if`, `if-else`, `if-elif-else`, nested conditions, input handling, type conversion |
| **3** | Conditionals (continued) | Logical operators, system libraries (`calendar`, `time`) |
| **4** | Iterations and Ranges | `while` loops, `for` loops, `range()`, `break`, `continue`, nested loops, formatted printing (f-strings) |
| **5** | Iterations (continued) | `math` and `random` libraries, recurrence relations, **functions** (arguments, scope, namespaces), **recursion** |
| **6** | Basic Collections | Lists — iteration, mutability, methods, nested lists, matrices, tuples, packing/unpacking |
| **8** | Collections (continued) | Dictionaries — iteration, methods, text processing with dicts, sets and set operations |
| **9** | File Operations | Opening, reading, writing files, file modes, `read`/`readline`/`readlines`/`write` |
| **10** | File Operations (continued) | CSV files, parsing structured data, files to collections |
| **11** | Module System | `import`, custom modules, standard library usage |

## 🚀 How to Use This Repository

**Requirements:** Python 3.7+ (files use the standard library only).

```bash
# Clone
git clone https://github.com/sujitsingh8/Python_Notes_IITM-BS.git
cd Python_Notes_IITM-BS

# Run any file
python "Week 4/example.py"
```

Each week's folder contains Python files that walk through the key concepts taught that week. Open them in any editor, read the code, and run snippets to see what happens.
